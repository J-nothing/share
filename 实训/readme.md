###五轴加工中心编程与操作技能

第一个实训主要是机床操作，50多个人分为两个班上课，每个班考的内容不同。自行前往实训基地。每天四次打卡，少一次记一整天缺勤，出勤要求不少于80%。

其中一个班真正考试的内容只讲了两天多一点，前面多数是铺垫。考试很简单，规范操作人不受伤绝对能过，机编程序会给参考，试卷内容都会讲，机床操作要求不严。

开课前上报了一部分需要参加外校夏令营或其他特殊情况的同学，这部分同学没有录入系统，签字单独签在一个表格上，系统没有打卡，考试没有准考证，最后也没有那个培训证书。最后考试的时候这部分人没有纸质试卷，有直接考试请假的，应该是没啥影响（仅限于这部分名单外的人）。

考核内容：1.机编程序；2.操作机床，运行给定程序完成工件的铣削；3.试卷（参考 五轴加工卷子（机床）.pdf）
